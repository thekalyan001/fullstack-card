
<h4>Setup project</h4>

   ```bash
   git clone https://github.com/thekalyan001/fullstack-card
   cd fullstack-card
   ```
   
<h4>Setup frontend</h4>

   ```bash
   cd frontend
   npm install
   npm run dev
   ```

<h4>Setup backend</h4>

   ```bash
   cd backend
   npm install
   npm start
   ```
